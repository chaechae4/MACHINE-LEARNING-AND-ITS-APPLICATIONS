# 🐶 얼굴상 분류 AI 프로그램 (강아지 vs 고양이 vs 공룡)

---

## 1. 프로젝트 개요
연에인 기사에서 자주 볼 수 있는 '00상'과 같이 생김새를 동물에 비유하는 문화가 대중화되었습니다.
요즘은 친구들 사이에서 이상형을 말할 때도 '강아지상이 좋아'와 같이 한 마디로 취향을 공유할 수 있을 만큼
보편적인 개념이 되었습니다. 이번 학기 동안 배운 기계학습 지식과 구글의 'Teachable Machine'을 활용하여,
누구나 자신의 얼굴상을 쉽게 판단할 수 있는 프로그램을 제작하고자 하였습니다.

- TensorFlow/Keras: 학습된 모델을 로드하고 실시간 데이터를 통해 결과를 예측합니다.  
- OpenCV & JS: 구글 코랩 환경에서 노트북 웹캠에 접근하여 실시간 이미지를 캡쳐합니다.

- Preprocessing: 입력된 이미지를 리사이징하고 -1 ~ 1 범위로 정규화합니다.

## 4. 보완할 점
- 환경 구축의 어려움: 프로젝트 초기, 로컬 pc에 설치된 아나콘다와 텐서플로우 등 다양한 라이브러리들이 가상환경
  내에서 충돌하고 버전 호환성 문제를 일으켜 실행에 어려움이 많았습니다. 이를 해결하기 위해 코랩으로 전환하여
  실행 문제를 해결했으나, 추후 로컬 환경의 의존성 관리 문제를 근본적으로 해결해야 할 필요성을 느꼈습니다.  
- data set의 한계: 학습 데이터 수집 시 좀 더 다양한 각도와 표정의 인물 사진을 충분히 사용하지 못한 점이
  아쉽습니다. 특히 특정 클래스(강아지)의 학습 이미지에 볼하트를 한 사진이 많아, 실제 테스트 시 인물이 볼하트
  포즈를 취하면 해당 클래스의 확률이 바정상적으로 높아지는 등 데이터 편향 현상이 확인됩니다.  
- 배경 노이즈 문제: 인물이 없을 때 발생하는 오작동을 줄이기 위해 배경 클래스를 추가하여 보완하였으나, 여전히 조명이나
  거리감에 따라 판정 결과가 불안정한 경우가 있어 더욱 정교한 추가 학습이 필요함을 깨달았습니다.
